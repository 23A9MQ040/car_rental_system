🚀 Quick Start

Prerequisites:

  • Java 17 JDK
  • Maven 3.6+
  • Git

Steps:

  1. Clone: git clone https://github.com/23A9MQ040/car_rental_system.git
  2. Navigate: cd car_rental_system
  3. Run: mvn clean spring-boot:run
  4. Open: http://localhost:8080
